# 练功房脚手架

如果你恰好也用Java做练功房的练习，可以从这个脚手架开始入手。

准备步骤：

* `git clone https://github.com/gigix/dojo-scaffold.git`
* `mv dojo-scaffold my_kata && cd my_kata`
* `mvn clean package`
